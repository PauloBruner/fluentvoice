1. Overview

FluentVoice is a Chrome extension designed to help users practice language pronunciation and translation. We respect your privacy and are committed to protecting your information.

2. Data Collection

FluentVoice does not collect, store, or share personal data.

Text entered by the user may be processed temporarily to:

  • Generate translations via third-party translation APIs.
  • Generate speech audio via Google Text-to-Speech API.

No text is permanently stored by FluentVoice servers.

3. Third-Party Services

FluentVoice may use:

  • Google Cloud Text-to-Speech API
  • Translation APIs

These services process text only to return the requested translation or audio.

4. Data Storage

FluentVoice does not store personal data, browsing history, or user identifiers.

5. Security

All communications with backend services are encrypted using HTTPS.

This Privacy Policy may be updated when necessary.
